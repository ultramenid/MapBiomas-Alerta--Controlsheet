Design Details

  • Centered card layout - Clean white card on dot-pattern background
  • No animations - Instant rendering, no fade-ins or transitions
  • Clear typography hierarchy - Title, subtitle, form labels
  • Green accent button (#86B897) with white text
  • Input fields - Subtle borders, clear focus states
  • Dark mode support - Card becomes dark gray, inputs darker
  • Responsive - Card shrinks on mobile, stays centered
  • Error states - Red border on invalid, inline error message

What stays the same:

  • Dot-pattern background (light & dark)
  • Logo in bottom left (outside card, visible on page)
  • Theme toggle functionality (but with fix for flickering)
